Question-1: What are the different ways to select an element in the DOM?

In JavaScript there are 6 different ways to select an element such as getElementById , getElementByClassName , getElementByTagName , querySelector , querySelectorAll and getElementByName

Question-2: What is the difference between innerHTML, innerText, and textContent

The difference between innerHTML , innerText and textContent is in innerHTML, it shows HTML content inside an element, including all HTML tags, in innerText, it shows the visible text content of an element, excluding hidden elements and in textContent, it shows raw text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation is used to when there have many child elements that need the same event handling

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the parents of the target element in the same nesting hierarchy till it reaches the outermost DOM element

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript i create a element by using document.createElement("which element i want to create"). Then for adding i use appendChild() and for removing i use removeChild()
